## About

A multipage static website that promotes the camp and points families toward
registration. It highlights the four learning tracks, the daily schedule, the
mentors, and safety information for parents. Built to be fast, mobile-friendly,
and easy to update.

## Pages

| File | Page |
| --- | --- |
| `index.html` | Home — hero, purpose, and quick links |
| `tracks.html` | The four learning tracks and their classes |
| `schedule.html` | Daily schedule, sports by day, and lunch menu |
| `mentors.html` | The professionals and leaders who teach |
| `safety.html` | Parent information, safety, and contact |
| `register.html` | Registration and volunteer/mentor sign-up |

## How to view or edit

No build tools or installation required — these are plain HTML files.

- **View locally:** double-click `index.html` to open it in any web browser.
- **Edit text:** open a file in any text editor (or edit it directly on GitHub
  using the pencil icon), change the wording, and save/commit.
- Keep all files together in the same folder so the links between pages work.

## How it's published

This site is hosted with **GitHub Pages**.
To update the live site, edit or re-upload the files and commit the change —
the published site refreshes automatically within a minute or two.

## Things to keep current

- **Registration link** — update in `register.html` and `index.html` if the
  Church Center URL changes.
- **Contact info** — Angel Laurencio's name and phone appear in the footer of
  every page and on the Safety and Register pages.
- **Schedule, tracks, and lunch** — update `schedule.html` and `tracks.html`
  if classes or menus change.
- **Photos** — real camp photos can be added to the hero once parent photo
  permissions are in place.

## Design

Colors, typography, and layout follow the FUNMER 2026 brand system
(navy, orange, and teal; the Fredoka + Inter typefaces; color-coded tracks).
Fonts load from Google Fonts, so an internet connection is needed to display
the exact typefaces.
